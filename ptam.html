💡 Como usar a HP 12C

2. Deixe UM campo vazio (o que você quer calcular)

3. Pressione "CALCULAR DESCONHECIDO"

4. Use CHS (+/-) para valores negativos (entrada, amort.)

📖 GUIA COMPLETO DAS TECLAS HP 12C — PARA IMOBILIÁRIA
n Número de Parcelas / Prazo
Armazena ou calcula o número de pagamentos. Em financiamento imobiliário, representa o prazo em meses. Para anos, multiplique por 12 antes.
EXEMPLO IMOBILIÁRIO:
360 → [n] = financiamento de 30 anos
Para calcular: deixe [n] vazio e pressione ENTER
i% Taxa de Juros Mensal
Armazena ou calcula a taxa de juros por período (mês). Para converter taxa anual: divida por 12 ou use a tecla g→12÷. MCMV Faixa 2: 6% a.a. = 0,4868% a.m.
EXEMPLO IMOBILIÁRIO:
0.4868 → [i%] = taxa mensal MCMV F2
6 → [g] → [i%] = converte 6% a.a. em mensal
PV Valor Presente — Valor Financiado
O valor que está sendo financiado hoje. Em imóveis = Valor do imóvel MENOS a entrada MENOS o subsídio MENOS o FGTS. É o saldo que vai para o banco.
EXEMPLO IMOBILIÁRIO:
Imóvel R$ 250.000 – entrada R$ 50.000
200000 → [PV] = valor financiado
PMT Parcela Mensal de Pagamento
O valor da prestação mensal. Na HP, o PMT sai negativo (pagamento = saída de dinheiro). Para o sistema PRICE, o PMT é fixo durante todo o prazo. Para SAC, use a calculadora dedicada.
EXEMPLO IMOBILIÁRIO:
Deixe vazio e pressione ENTER para
calcular a parcela do financiamento
FV Valor Futuro — Saldo Devedor Final
O valor que restará ao final do prazo. Em financiamentos normais, FV = 0 (dívida zerada). Útil para calcular o saldo devedor após X meses ou o valor do balão residual.
EXEMPLO IMOBILIÁRIO:
0 → [FV] = financiamento total quitado
Deixe vazio para calcular saldo devedor
f Shift Amarelo — Funções Financeiras
Ativa as funções secundárias escritas acima das teclas em amarelo. Principal uso: f → AMORT (amortização acumulada), f → CLx (limpar registros financeiros).
COMBINAÇÕES ÚTEIS:
[f] + [CLx] = zera todos os registros
[f] + [PMT] = calcula amortização acumulada
g Shift Azul — Funções Matemáticas
Ativa funções escritas abaixo das teclas em azul. Fundamental para converter taxas anuais em mensais (12÷) e mensais em anuais (12×). Também controla modo BEGIN/END.
COMBINAÇÕES ÚTEIS:
[g] + [i%] = 12÷ (anual → mensal)
[g] + [n] = 12× (anos → meses)
STO Guardar Valor na Memória
Salva o número mostrado no display em uma das memórias (0 a 9). Útil para guardar resultados intermediários durante cálculos complexos de financiamento.
EXEMPLO:
Calcula parcela → [STO] → [1]
O valor fica salvo para uso posterior
RCL Recuperar Valor da Memória
Recupera um valor guardado com STO. [RCL] + [n] mostra o número de parcelas registrado. [RCL] + [PV] mostra o valor financiado. Muito útil para revisar os dados inseridos.
EXEMPLO:
[RCL] + [PV] = mostra valor financiado
[RCL] + [i%] = mostra taxa inserida
CLx Limpar Display / Apagar Erro
Limpa o número atual no display sem apagar os registros financeiros. Use quando digitar um número errado. Para limpar TUDO (n, i, PV, PMT, FV), use [f] + [CLx].
DICA IMPORTANTE:
[CLx] = apaga só o display
[f]+[CLx] = apaga TODOS os registros
CHS Inverter Sinal (+/−)
Inverte o sinal do número no display (positivo→negativo ou vice-versa). Na HP 12C, pagamentos (PMT) e entrada (PV em algumas convenções) devem ser negativos para o cálculo funcionar corretamente.
QUANDO USAR:
PMT sai negativo = é pagamento (saída)
50000 [CHS] → [-50000] = entrada paga
÷ Divisão — Calcular Taxa Proporcional
Divisão comum. No contexto imobiliário: use para converter taxa anual em mensal manualmente (taxa a.a. ÷ 12), ou calcular percentual de entrada (entrada ÷ valor do imóvel × 100).
EXEMPLO:
8.16 [ENTER] 12 [÷] = 0,68% a.m.
50000 [ENTER] 250000 [÷] = 20% entrada
× Multiplicação — Total Pago / Custo Real
Multiplicação. Fundamental para calcular o custo total do financiamento: PMT × n = total pago. Compare com o valor financiado para ver quanto os juros custam no total.
EXEMPLO:
1500 [ENTER] 360 [×] = R$ 540.000
= total pago em 30 anos de R$ 1.500/mês
+ Adição — Valor Total do Imóvel
Adição simples. Use para somar: entrada + FGTS + subsídio para calcular o total da "entrada composta". Ou para calcular custo total: valor financiado + juros totais pagos.
EXEMPLO:
20000 [ENTER] 15000 [+] = R$ 35.000
= entrada (R$20k) + FGTS (R$15k)
− Subtração — Valor Financiado Líquido
Subtração simples. Use para calcular o valor financiado: valor do imóvel − entrada − subsídio − FGTS = PV (o que vai para o banco financiar).
EXEMPLO:
250000 [ENTER] 50000 [−] 29000 [−]
= R$ 171.000 (valor a financiar)
📋 SEQUÊNCIA PADRÃO — CALCULAR PARCELA MCMV

PASSO A PASSO NA HP 12C REAL

1. Limpar registros: [f] + [CLx]
2. Prazo em meses: 360 → [n]
3. Taxa mensal: 6 → [g] → [i%] (converte 6% a.a. em mensal)
4. Valor financiado: 200000 → [PV]
5. Saldo final: 0 → [FV]
6. Calcular parcela: [PMT] (resultado aparece negativo = pagamento)
